# shift4-recruitment-donation-form
Working on node version `20.10.0`

## How to run

Install packages (yarn version 1)
```
yarn install --frozen-lockfile
```

Run application (using `--turbo` from next.js 14)
```
yarn dev
```

## Testing
### E2E

```
yarn build
yarn test:e2e
```
### Unit

```
yarn test:unit
```
### Both

```
yarn build
yarn test
```
